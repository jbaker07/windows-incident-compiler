SECTION A — ONE-SENTENCE TRUTH
================================================================================

The product is a Windows EDR incident compiler that captures Windows Event Log
telemetry via ETW, extracts canonical facts (Exec, ProcSpawn, OutboundConnect,
etc.), matches them against YAML playbook slot predicates using a hypothesis
engine, promotes complete hypotheses to signals with full explainability, and
serves results via HTTP API with browser-based UI.
